Install

sudo bash install-support.sh
  1. Download install-support.sh.
  2. Open a terminal in the download location.
  3. Run the install command above.
  4. After install, use the desktop icons for day-to-day support access.

When you actually need support

Most users should just double-click the desktop icon:

  • Enable Remote Support to turn support on
  • Disable Remote Support to turn support off

If support is being enabled, the script will prompt for your Bedrock support key. We provide that key separately when support is needed.

Terminal fallback: sudo support-on.sh

Optional guided version: sudo support-on.sh --support-key '<key from Bedrock>'
